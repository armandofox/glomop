Try a GloMop Technology Demo!

If you surf the Web using a conventional modem (28.8 or slower) and you are not behind a security firewall (i.e. your browser allows you to designate an arbitrary host as an HTTP proxy), this technology demo will show you how the ideas behind GloMop can be implemented to make Web surfing bearable even with a slow modem.
